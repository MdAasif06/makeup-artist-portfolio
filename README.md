# 💄 Makeup Artist Portfolio Website

A modern and responsive **Makeup Artist Portfolio Website** built using **React.js, Vite, and Tailwind CSS**.
This project is designed to showcase makeup services, pricing, and contact details in a clean and elegant way.

---

## 🚀 Features

* 🌐 Multi-page layout (Home, About, Services, Contact)
* 🎨 Modern UI with Tailwind CSS
* 📱 Fully responsive (Mobile, Tablet, Desktop)
* 💼 Professional portfolio design
* 💌 Contact form section
* 💅 Beautiful service & pricing cards
* ⚡ Fast performance using Vite

---

## 🛠️ Tech Stack

* React.js
* Vite
* Tailwind CSS
* JavaScript (ES6+)

## ⚙️ Installation & Setup

1. Clone the repository:

```
git clone https://github.com/your-username/makeup-artist-portfolio.git
```

2. Navigate to project folder:

```
cd makeup-artist-portfolio
```

3. Install dependencies:

```
npm install
```

4. Run the development server:

```
npm run dev
```
