# Nilofer Beauty Portfolio

A modern, responsive **Makeup Artist Portfolio Website** built with **Vite**, **React**, and **Tailwind CSS**.

## ✨ Features

- Elegant beauty-brand inspired UI
- Responsive `Home`, `About`, `Services`, and `Contact` pages
- Reusable components for navbar, footer, and service cards
- Soft pink / beige / white palette
- Smooth hover states and subtle motion effects
- Contact form UI for lead generation

## 📁 Folder Structure

```text
src/
├── assets/
├── components/
├── data/
├── pages/
├── App.jsx
├── index.css
└── main.jsx
```

## 🚀 Run the Project

```bash
npm install
npm run dev
```

Then open the local URL shown in the terminal, usually:

```bash
http://localhost:5173/
```

## 🏗️ Build for Production

```bash
npm run build
```
